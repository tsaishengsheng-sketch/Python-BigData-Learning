# 🔀 Logic Control (邏輯判定練習)

本單元練習如何讓程式根據不同條件執行任務，掌握基本的邏輯分歧處理。

### 📝 學習重點
* **條件分支應用**：使用 `if-elif-else` 處理多重選擇，例如 `grade_level_check.py` 中的分數等級判定。
* **實務邏輯判定**：實作閏年判定、星座計算、以及十六進位轉換等基礎數學邏輯。
* **輸入驗證與防呆**：練習基本的零值檢查與除法除數檢查，確保程式執行時不會報錯。
* **基礎互動遊戲**：實作猜數字遊戲的判斷邏輯，學習如何處理使用者輸入的大小判定。

### 📂 檔案結構
1.  **歸檔練習**：位於 [`learning/`](./learning/) 資料夾，包含所有 .ipynb 格式的邏輯練習題。
2.  **核心整合**：[`SmartLife_Toolbox.py`](./SmartLife_Toolbox.py) - 將多個邏輯判斷功能整合在一起的工具。
