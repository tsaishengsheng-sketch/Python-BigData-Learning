# 📦 Functions (函式模組化與系統整合)
學習將複雜的邏輯封裝，提高程式碼的重複使用性與可讀性。

### 📝 學習重點
* **定義與呼叫**：設計 `def` 關鍵字與 `return` 回傳值，實現模組間的資料傳遞。
* **參數傳遞**：處理使用者輸入的外部數據（如：體位、成績、座標、氣溫）。
* **邏輯抽象化**：將幾何運算（多邊形面積）與統計邏輯（計票、平均值）封裝，保持主程式簡潔。
* **安全與權限連動**：實作驗證碼比對（Random）與等級門檻（GPA），模擬真實軟體的權限控管。

---

### 🌟 綜合實作專案：智慧年度評選系統 (Smart Award System)
將 15+ 個碎片化練習題，整合為一套具備「行政管理與安全檢核」流程的連貫系統。



#### 🔧 系統整合邏輯：
1. **身分初始化 (BMI, Temp, Random)**：
   輸入身高體重與攝氏溫度，系統同步生成「6 位數投票驗證碼」供後續安全核對。
2. **績效資格審查 (Sum/Avg, GPA)**：
   計算考核平均分並判定等第。**連貫邏輯：** 僅 A 級績效者能進入頒獎與投票流程。
3. **場地規劃 (Polygon Area, Distance)**：
   根據使用者輸入參數，精算頒獎台面積與會場座標位移距離。
4. **安全投票機制 (Election Simulator)**：
   強制核對第一階段生成的隨機驗證碼，正確後方可執行最終計票表決。

---

### 📂 檔案結構
* **Smart_Award_System.py**：核心整合作品，展示函式間的邏輯連動。
* **learning/**：基礎練習存檔（包含 BMI、GPA、溫標、多邊形、距離、計票等原始練習碼）。
