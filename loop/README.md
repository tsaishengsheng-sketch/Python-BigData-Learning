# 🔁 Loops (迴圈控制)
本單元透過 25+ 個練習題掌握重複執行邏輯，並將其整合為功能選單。

### 🌟 綜合實作：[Loop_Practice_Tool.py](./Loop_Practice_Tool.py)
將本單元零散的練習整合為一個具備選單的工具，包含：
* **規律生成**：九九乘法表、數字金字塔。
* **數學判定**：質數 (Prime) 檢查與 GCD (最大公約數) 計算。
* **演算法實作**：手寫選擇排序 (Selection Sort) 與隨機樂透產生器。

### 📂 檔案結構
1. **Loop_Practice_Tool.py**：本單元的綜合實作作品。
2. **learning/**：包含九九乘法、猜數字、排序等基礎練習的 `.ipynb` 存檔。
