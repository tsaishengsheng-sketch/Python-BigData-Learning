# 📐 Variable Constant (地產投資分析與貸款壓力測試)
本單元整合基礎數學運算與財務公式，開發出一套能對比不同還款方案、計算土地面積與校驗產權的決策工具。

### 📝 學習重點
* **數位拆解與校驗邏輯**：利用 `%` 與 `//` 運算子進行多位數拆解，實作資料校驗還原演算法。
* **三角幾何土地估價**：應用 `math.radians` 與 `math.sin` 處理現實世界的不規則圖形面積計算。
* **雙方案本息攤還對比**：使用 `pow()` 函數實作複利模型，精確計算不同年限下的財務負擔。
* **利息節省率決策分析**：透過邏輯對比運算，量化提前還款所產生的購屋成本節省率。

---

### 🌟 綜合實作專案：RealEstate_Investment_Analyzer
具備「還款方案對比」功能的地產分析器，能精確計算 40 年與 20 年貸款的利息差異，協助進行資金成本優化。

#### 🔧 系統整合邏輯：
1. **地籍編號反轉校驗**：模擬舊式資料庫的還原機制，透過整數除法逐位提取數字，驗證文件編號真偽。
2. **弧度與正弦面積運算**：將使用者輸入的「角度」轉換為電腦運算所需的「弧度」，並利用正弦定理公式 $Area = 0.5 \times a \times b \times \sin(\theta)$ 算出土地面積。
3. **多組複利公式對比運算**：建構本息平均攤還模型 $$MonthlyPay = \frac{P \times r \times (1+r)^n}{(1+r)^n - 1}$$ 並同步計算長年限與短年限的利息總額。
4. **格式化輸出節省成本報告**：計算兩方案之利息差額 (Saved Money)，並將其轉化為相對於本金的「購屋折扣率」，以格式化字串呈現直觀的決策建議。
