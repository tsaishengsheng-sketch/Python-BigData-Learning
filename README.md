# 🐍 Python Big Data Learning

課程實作紀錄：蔡明志老師《Python大數據》練習題彙整
 
## 📂 專案導覽

本專案包含《Python大數據》課程的所有實作練習，目前正進行格式最佳化中。

| 單元目錄 | 內容說明 | 關鍵技術 |
| :--- | :--- | :--- |
| **variable_constant** | 基礎語法與變數運算 | 變數運算、幾何公式 |
| **if_else** | 選擇結構與條件分支 | 邏輯判斷、例外處理 |
| **loop** | 迭代結構與演算法 | 質數搜尋、排序演算法 |
| **function** | 程式模組化與封裝 | 函式定義、參數傳遞 |
| **list** | 串列操作與搜尋、排序 | 氣泡排序、二元搜尋 |
| **utils** | 工具程式 | 通用 Function 封裝 |

## 🛠️ 開發工具

- **Environment**: macOS M2 Air / VS Code
- **Control**: Git / GitHub CLI
- **Status**: Converting scripts to Jupyter Notebooks (.ipynb)
